Números enteros

Números en coma flotante

Números enteros y en coma flotante permiten las mismas operaciones.

¿Por que nos importa la diferencia?

  • Unos datos como 'número de ruedas de un vehículo', 'número de clientes por día' o 'número de compras en una página de web' son mejor representados por números enteros para garantía de calidad.
  • Números enteros son precisos, mientras los en coma flotante son imprecisos. Es una diferencia técnica que veremos en la próxima diapositiva, pero volveremos a verla más adelante cuando usemos numpy y pandas

Ejemplo de imprecisión numérica

¿Por qué pasa eso? En nuestro sistema de números del día cotidiano, que sigue la base de diez, dividir 1 entre 3 nunca nos da un resultado preciso. En una manera parecida, los números de las computadoras, que siguen una base de dos (binario), tienen muchos más números imprecisos.

Cadena de caracteres

Cuando usamos cadena de caracteres, concatenamos con '+' y repitimos con '*'. Para asignar una cadena de caracteres que cabe más de una linea, usamos tres comillas o \n (newline, o sea 'nueva linea'). Se puede remplazar palabras usando .replace(palabra_no_deseada, nueva_palabra)

Tipo booleano

Los booleanos se usan para guardar información que es verdadero o falso como 'tiene_iphone' (tienes un iphone, o no; son las únicas opciones) o para comprobar condiciones lógicas. >En la Parte 2 aprendimos como manejar pedacitos de información aislados. Ahora vamos a construir formaciones más complejas, agrupando estos tipos básicos dentro de los envases que brinda Python. Los siguientes envasos son los más básicos, pero también los más importantes y frecuentes

Resumen

Nombre En Inglés Uso Ejemplos
lista list colección de cosas, como una bolsa donde se mete todo x = [1, 2, 2, 3]
y = list([1, 2, 3])
diccionario dictionary relaciona claves con valores; la clave debe ser una cadena de caracteres, el valor cualquier cosa x = {"primer_numero": 1, "segundo_numero": 2}
y = dict(primer_numero=1, segundo_numero=2)
conjunto set Un conjunto de matemática, cada elemento es único. y = type({1, 2})

tupla tuple Como una lista, pero no se puede modificar después. x = (1,)

No profundizaremos mucho en tuplas porque apenas se usan en análisis de datos.

Listas

.append() agrega un elemento; .insert() insierta un elemento en el índice especificado; .extend() agrega otra lista; .reverse() invierte; lista[2] devuelve/reasigna el elemento con índice 2; lista[2:4] devuelve/reasigna los elementos con índice 2 y 3, pero sin elemento con índice 4; len() da el número de elementos dentro de una lista.

Diccionarios

len() da el número de pares; .get("clave") devuelve el valor de "clave"; .pop("clave") es como .get("clave") pero también borra el par del diccionario; .values() devuelve todos los valores; .keys()todos los claves; .items() todos los pares en una lista de tuples; diccionarios se pueden concatenar usando |.

Conjuntos

.union() une elementos únicos de dos conjuntos; .intersection() identifica elementos compartidos; .difference() identifica elementos que están en el primer conjunto pero no en el segundo
